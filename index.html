1. Project Description

이 프로젝트는 HTML과 CSS의 기본 개념을 이해하고 연습하는 목적으로 제작되었다.
CSS를 이용한 HTML 스타일링 외에 Form page를 제작함으로서 Form을 이해하고 연습한다.

2. Project Information

이 프로젝트는 HTML5 마크업 언어를 사용하였으며,
소스코드 편집기로는 1.82.2 (Universal)버전의 Visual Studio Code를 사용하였다.

또한 Ventura 13.4버전의 macOS에서 제작되었으며,
117.0.5938.92버전의 Google Chrome을 기반으로 제작되었다.
